## Usage Scenarios

### Scenario 1: Presentation Mode

```vue
<!-- High contrast for projectors -->
<UnaEditor theme="dark" code-theme="dracula" />
```

**Why Dracula?** Vibrant colors show well on projectors

### Scenario 2: Documentation Site

```vue
<!-- Clean, professional look -->
<UnaEditor theme="light" code-theme="github-light" />
```

**Why GitHub Light?** Familiar to developers, clean

### Scenario 3: Night Coding

```vue
<!-- Easy on eyes at night -->
<UnaEditor theme="dark" code-theme="nord" />
```

**Why Nord?** Low contrast, cool tones reduce eye strain

### Scenario 4: Long Reading Sessions

```vue
<!-- Scientifically designed for comfort -->
<UnaEditor theme="light" code-theme="solarized-light" />
```

**Why Solarized Light?** Designed for reduced eye fatigue

### Scenario 5: Mixed Content

```vue
<!-- Dark editor, light code blocks for contrast -->
<UnaEditor theme="dark" code-theme="github-light" />
```

**Why Mix?** Code blocks stand out from prose

## Color Psychology

### Dracula (Purple/Pink)

- **Feeling**: Creative, energetic
- **Use case**: Creative coding, late night sessions
- **Audience**: Designers, frontend developers

### Monokai (Warm tones)

- **Feeling**: Comfortable, nostalgic
- **Use case**: Long coding sessions
- **Audience**: Sublime Text users, backend developers

### Nord (Cool blues)

- **Feeling**: Calm, focused
- **Use case**: Concentration work
- **Audience**: Minimalists, Scandinavian design fans

### Solarized (Balanced)

- **Feeling**: Scientific, precise
- **Use case**: Technical writing, research
- **Audience**: Scientists, technical writers

### Tokyo Night (Modern)

- **Feeling**: Trendy, sleek
- **Use case**: Modern web development
- **Audience**: Frontend developers, UI designers

## Accessibility Considerations

### Contrast Ratios

| Theme           | WCAG AA | WCAG AAA       | Notes                              |
| --------------- | ------- | -------------- | ---------------------------------- |
| One Dark        | ✅ Pass | ✅ Pass        | Good contrast                      |
| Dracula         | ✅ Pass | ⚠️ Some colors | Pink may be too bright             |
| Monokai         | ✅ Pass | ✅ Pass        | Excellent contrast                 |
| Solarized Dark  | ⚠️ Low  | ❌ Fail        | Designed for comfort, not contrast |
| Nord            | ⚠️ Low  | ❌ Fail        | Low contrast by design             |
| Tokyo Night     | ✅ Pass | ✅ Pass        | Good contrast                      |
| GitHub Light    | ✅ Pass | ✅ Pass        | Excellent contrast                 |
| Solarized Light | ⚠️ Low  | ❌ Fail        | Designed for comfort               |
| Atom One Light  | ✅ Pass | ✅ Pass        | Good contrast                      |

**Note**: Solarized and Nord intentionally use lower contrast for reduced eye strain. They may not meet WCAG AAA standards but are designed for long-term comfort.

## Theme Recommendations by Language

### JavaScript/TypeScript

- **Best**: Dracula, One Dark, Tokyo Night
- **Why**: Good distinction between keywords, types, and values

### Python

- **Best**: Monokai, One Dark
- **Why**: Clear string highlighting, good for indentation

### CSS

- **Best**: GitHub Light, Atom One Light
- **Why**: Clean property/value distinction

### Go

- **Best**: Nord, Tokyo Night
- **Why**: Minimalist languages match minimalist themes

### PHP

- **Best**: Monokai, Dracula
- **Why**: Good for mixed HTML/PHP content

### Shell/Bash

- **Best**: Solarized Dark, Nord
- **Why**: Terminal-like feel
